#### 1,Wepy
 1, 安装
 $ npm install @wepy/cli -g # 全局安装 WePY CLI 工具
 $ wepy init standard myproj # 使用 standard 模板初始化项目
 $ cd myproj # 进入到项目目录
 $ npm install # 安装项目依赖包
 $ npm run dev # 监听并且编译项目
